# US-0001: 図書館利用者として登録しログインを開始できる

## Status
Todo

## User Story
As a 図書館利用者,  
I want メールアドレスとパスワードで利用登録したい,  
so that サービスにログインして本の探索や予約機能を使い始められる.

## User Benefit
登録とログイン開始までの手順を最小化することで、図書館利用者が迷わずサービス利用を開始できる。ログイン可能になることで、本の探索や予約といった価値提供機能へスムーズに到達できる。

## Acceptance Criteria
- [ ] 未登録のメールアドレスと有効なパスワードを入力すると、登録が完了する。
- [ ] 登録成功後、同一の認証情報で直ちにログインできる。
- [ ] 既に登録済みのメールアドレスでは登録できず、重複を示すエラーメッセージが表示される（HTTPステータスは `400`）。
- [ ] 必須項目（メールアドレス、パスワード）が未入力の場合は登録できず、項目ごとのエラーが表示される。
- [ ] メールアドレスに半角以外の文字が含まれる場合、またはパスワード要件未満の場合は登録できず、入力修正に必要なエラーが表示される。
- [ ] 登録失敗時はアカウントが作成されない。

## Non-Goals
- メール認証フローの導入
- 管理者承認フローの導入
- プロフィール拡張項目（表示名、嗜好タグなど）の登録
- OAuth/SNSログイン対応
